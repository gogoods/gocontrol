An usefull script for controling golang program written by shell, also can be used for other binary application.


### Features

- [x] ./control status
- [x] ./control start
- [x] ./control stop
- [x] ./control restart
- [x] ./control build
- [x] ./control pack
- [x] ./control release
- [x] ./control tail
- [x] ./control monitor

### Usage

```
goctrl <appname>
```
